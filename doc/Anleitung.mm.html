~ Anleitung

  • Einstellungen

    • Generell wird unterschieden zwischen permanenten Einstellungen - wie Darstellungsoptionen und Spracheinstellungen (Datei, Grafik, Substitution), die während der Laufzeit nicht verändert werden und veränderlichen Einstellungen - wie Suchoptionenen - die während der Laufzeit geändert werden können

      Permanente Einstellungen

      • Die Konfigurationsdatei (config.cfg) kann mit einem beliebigen Texteditor verändert werden. Alle Konfigurationsoptionen sind durch Kommentare erläutert.

      Veränderliche Einstellungen

      • Veränderliche Einstellungen können während einer Sucheeingabe durch Tastenkombinationen verändert werden durch die entsprechende Tastenkombination ALT+...

  • Bedienung
    • Hauptfenster

      • Das Hauptfenster wird nur dann sichtbar, wenn die in der Konfigurationsdatei definierte Tastenkombination erkannt wird.

        Sobald das Hauptfenster aktiv ist, kann ein Suchstring eingegeben werden und mit Return eine Suche durchgeführt werden.

        Wird nach einer Suche ein zweites Mal Return eingegeben, wird der in der Sprachkonfiguration definierte Suchstring im Browser geöffnet.

        Werden keine oder sehr wenige Fundstellen gefunden, dann wird eine Annäherungssuche durchgeführt, sofern diese Option aktiviert ist.

        Das Hauptfenster wird geschlossen, wenn

        • Die Tastenkombination erneut betätigt wird.
        • Ein anderes Fenster ausgewählt wird
        • ESC gedrückt wird bei einer leeren Suchanfrage (andernfalls löscht ESC den Suchstring)

        • Dateiname der Bitmap zur Darstellung (rechts oben)

        • Such-URL für Online-Suche (in <url> wird "<SEARCH>" mit dem zu suchenden Begriff ausgewechselt
      • DELETECOMMENT()
        • Löscht alle Zeilen die mit '#' beginnen
      • DELETESTRING(<str>)
        • Löscht einzelne Strings <str> zB {noun}
      • REPLACECHAR(<in>,<repl>)
        • Ersetzt einzelne Zeichen; kann zB verwendet werden um aus Tabulator \t Leerzeichen zu generieren
      • SUBSTITUTE(<inchar>,<outstr>)

        • Ersetzt einzelne Zeichen inchar mit einem String outstr; kann zB verwendet werden, um Accents etc automatisiert zu suchen
